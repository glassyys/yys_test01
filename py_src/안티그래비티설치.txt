Developer: Reload Window 해도 계속되는데 분석을 안하게 하는 방법과 분석을 다시 하게 하는 명령

창을 다시 로드해도 로딩 루프가 계속되는 것은 안티그래비티의 파이썬 분석 도구(Pyrefly나 기본 언어 서버)가 해당 파일의 구문이나 라이브러리를 해석하느라 멈추지 않는 현상입니다.

이 분석 작업을 강제로 멈추는 방법(안 하게 하는 방법)과 필요할 때 다시 분석을 시작하게 하는 명령(초기화)을 정리해 드립니다.

1. 분석을 일시적으로 안 하게 하는 방법 (비활성화)
가장 직관적인 방법은 하단 상태 표시줄에 꼬여있는 파이썬 분석 인터프리터를 '선택 해제(Clear)'하여 분석할 대상 자체를 없애버리는 것입니다.

단축키 Ctrl + Shift + P를 누릅니다.

입력창에 Python: Clear Workspace Interpreter Setting을 입력하고 선택합니다.

이렇게 하면 파이썬 지정을 취소하므로, 하단 바가 다시 처음 보셨던 노란색 ⚠️ Select Interpreter 상태로 돌아가며 백그라운드 분석 연산을 즉시 중단합니다.

💡 코드 검사(Linting) 기능만 끄고 싶다면?
실시간 문법 분석 기능만 끄고 싶다면 Ctrl + Shift + P를 누르고 **Python: Enable Linting**을 검색한 뒤, 설정을 **Off**로 바꾸시면 분석 작업을 하지 않습니다.

2. 분석을 강제로 다시 하게 하는 명령 (재시작)
무한 루프에 빠진 분석 엔진을 완전히 청소하고 처음부터 깨끗하게 다시 분석을 시도하게 만드는 명령입니다. 창 전체를 닫았다 켜는 것보다 파이썬 엔진만 리셋하므로 효율적입니다.

단축키 Ctrl + Shift + P를 누릅니다.

입력창에 Python: Restart Language Server (파이썬 언어 서버 재시작)을 입력하고 엔터를 누릅니다.

이 명령을 실행하면 현재 꼬여있던 분석 프로세스가 강제 종료된 후, 깨끗한 상태에서 소스코드 분석을 처음부터 다시 시작합니다.
