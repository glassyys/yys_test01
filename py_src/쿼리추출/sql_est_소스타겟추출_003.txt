아래 원본소스에서 파일생성전 
crud_type,abs_path,file,full_path 항목은 동일하게 생성하면서
추출된 queries 의 queries.append(query) 으로 처리된 최종 쿼리의 
MAIN_QUERY_START값의 키워드에서  ";"까지의 내용에서 아래 로직 반영하여 
소스와 타겟테이블정보를 추출하는 로직을 기존 로직 최대한 출력문 구조도 유지해서 수정하여 csv 파일로 생성하는 전체소스
1. 키워드 기준 queries(queries.append(query)) 내용에 대해서 SQL구조를 분석하여
MAIN_QUERY_START의 값은 sql_typ항목과 source_table, target_table 구분해서 csv파일을 생성하는 로직으로 변경
2. target_table에 대한 source_table은 여러행이 생성될수 있는 구조로 처리하여 target_table은 중복해서 생성되어도 됨
3. 최종 csv레이아웃
crud_type,abs_path,file,full_path,sql_typ,source_table,target_table
4.csv파일 구분자는 ","로 생성
5. 키워드 기준 queries(queries.append(query)) 내용 분석시 아래내용 반영
1) 서브쿼리 내부 테이블 추출 강화
2) WITH 절 분석
3) CTAS 구조 완전 분석
4) MERGE USING 서브쿼리 완전 파싱

6. 기존 출력부분은 출력가능한 부분 유지

7. FROM aa_tb a, 
        bb_tb b 인 경우
   FROM 절 이후 구간을 잘라서 "," 기준으로 분리하여
   aa_tb, bb_tb 모두 추출되도록 수정
단, SELECT 절의 컬럼 , 구분은 절대 추출되지 않도록
→ FROM ~ (WHERE/GROUP/ORDER/...) 까지만 분석

8.DELETE 
  FROM tb_1 a
tb_1 이 target_table 로 정상 기록되도록 수정

9. "END", "COMMIT"이나 
테이블이 될 수 없는 아래 예약어를 소스 타겟에 사용하지 말고 로직 구현
    "SET", "WHERE", "AND", "OR", "ON", "WHEN",
    "THEN", "ELSE", "VALUES", "SELECT",
    "UPDATE", "INSERT", "DELETE", "MERGE",
    "USING", "FROM", "JOIN", "INTO",
    "GROUP", "ORDER", "BY", "HAVING"
타겟테이블은 insert 나 upadate나  merge 같은경우이고
소스테이블은 from, join, using 뒤에서 사용하는 쿼리 규칙에 맞는 테이블값을 추출

10. "FROM DUAL"으로 구성된 단독쿼리와 "ALTER"키워드로 시작되는 경우는 테이블이 없으면 소스테이블 및 타켓추출은 안해도 되고
"ALTER TABLE" 와 "TRUNCATE TABLE" 다음에 있는 테이블은 타겟테이블로 추출되어야 함

#!/usr/bin/env python3
# 실행방법:
# python3 프로그램명.py 절대경로포함소스디렉토리

import os
import re
import sys
from datetime import datetime

# ==============================
# 1. 파라미터 체크
# ==============================

if len(sys.argv) != 2:
    print("사용법: python3 프로그램명.py 절대경로포함소스디렉토리")
    sys.exit(1)

SOURCE_DIR = os.path.abspath(sys.argv[1])

if not os.path.isdir(SOURCE_DIR):
    print("오류: 유효한 디렉토리가 아닙니다.")
    sys.exit(1)

# ==============================
# 2. 설정
# ==============================

TARGET_EXTENSIONS = ('.sh', '.hql', '.sql', '.uld', '.ld')
DELIMITER = "^"

EXCLUDE_PATTERNS = [
    "insert into sidtest.ad1901_rgb_ac190212_svc(svc_mgmt_num)"
]

# FROM DUAL 단독 쿼리만 제외
ONLY_FROM_DUAL_PATTERN = re.compile(
    r'^\s*SELECT\s+.*?\s+FROM\s+DUAL\s*;?\s*$',
    re.IGNORECASE | re.DOTALL
)

# ==============================
# 3. 전처리
# ==============================

def preprocess(content):

    content = "\n".join(
        line for line in content.splitlines()
        if not line.lstrip().startswith("#")
    )

    content = "\n".join(
        line for line in content.splitlines()
        if not re.match(r'(?i)^\s*DBMS_OUTPUT', line)
    )

    content = "\n".join(
        line for line in content.splitlines()
        if not (line.strip().startswith("/*") and line.strip().endswith("*/"))
    )

    pattern = re.compile(
        r"""
        ('([^']|'')*') |
        ("([^"]|"")*") |
        (--[^\n]*$) |
        (/\*.*?\*/)
        """,
        re.MULTILINE | re.DOTALL | re.VERBOSE
    )

    def replacer(match):
        if match.group(1) or match.group(3):
            return match.group(0)
        return ""

    return pattern.sub(replacer, content)

# ==============================
# 4. MAIN QUERY 시작
# ==============================

MAIN_QUERY_START = re.compile(
    r"""
    \b(
        CREATE\s+TEMPORARY\s+VIEW|
        CREATE\s+VIEW|
        CREATE\s+TABLE|
        ALTER\s+TABLE|
        DROP|
        TRUNCATE|
        DECLARE|
        BEGIN|
        EXECUTE|
        COMMIT|
        END|
        REPLACE\s+VIEW|
        MERGE|
        UPSERT|
        INSERT|
        UPDATE|
        DELETE|
        SELECT
    )\b
    """,
    re.IGNORECASE | re.VERBOSE
)

END_IF_PATTERN = re.compile(r'^\s*END\s+IF\b', re.IGNORECASE)

# ==============================
# 5. 쿼리 추출
# ==============================

def extract_queries_from_file(file_path):

    queries = []

    try:
        with open(file_path, 'r', encoding='utf-8', errors='ignore') as f:
            content = f.read()

        content = preprocess(content)

        pos = 0
        length = len(content)

        while pos < length:

            match = MAIN_QUERY_START.search(content, pos)
            if not match:
                break

            start = match.start()
            keyword = match.group(1).upper()

            # END IF 제외
            if keyword == "END":
                line_start = content.rfind("\n", 0, start) + 1
                line_end = content.find("\n", start)
                if line_end == -1:
                    line_end = length

                line_text = content[line_start:line_end]

                if END_IF_PATTERN.match(line_text):
                    pos = line_end
                    continue

            # COMMIT 제외
            if keyword == "COMMIT":
                end = start
                while end < length and content[end] != ";":
                    end += 1
                pos = end + 1
                continue

            # 세미콜론까지 추출
            end = start
            in_string = False
            quote_char = None

            while end < length:
                ch = content[end]

                if ch in ("'", '"'):
                    if not in_string:
                        in_string = True
                        quote_char = ch
                    elif quote_char == ch:
                        in_string = False

                if ch == ";" and not in_string:
                    end += 1
                    break

                end += 1

            query = content[start:end].strip()

            if query:
                lower_query = query.lower()

                if any(p.lower() in lower_query for p in EXCLUDE_PATTERNS):
                    pos = end
                    continue

                # FROM DUAL 단독 SELECT만 제외
                if ONLY_FROM_DUAL_PATTERN.match(query.strip()):
                    pos = end
                    continue

                queries.append(query)

            pos = end

    except Exception as e:
        print(f"[ERROR] 파일 읽기 실패: {file_path} ({e})")

    return queries

# ==============================
# 6. CRUD 분류
# ==============================

def classify_query(query):
    q = query.strip().upper()

    if q.startswith(("CREATE", "INSERT", "MERGE", "UPSERT",
                     "DECLARE", "BEGIN", "EXECUTE", "END")):
        return "C"
    elif q.startswith("SELECT"):
        return "R"
    elif q.startswith("UPDATE"):
        return "U"
    elif q.startswith("DELETE"):
        return "D"
    elif q.startswith(("ALTER", "REPLACE", "DROP", "TRUNCATE")):
        return "U"
    else:
        return "R"

# ==============================
# 7. 메인
# ==============================

def main():

    program_name = os.path.splitext(os.path.basename(sys.argv[0]))[0]
    source_last_dir = os.path.basename(SOURCE_DIR.rstrip(os.sep))

    out_dir = os.path.join(os.getcwd(), "out")
    os.makedirs(out_dir, exist_ok=True)

    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    output_filename = f"{program_name}_{source_last_dir}_{timestamp}.dat"
    output_path = os.path.join(out_dir, output_filename)

    total_files = 0
    total_queries = 0
    total_output_rows = 0
    total_file_lines = 0

    with open(output_path, 'w', encoding='utf-8') as out_file:

        for root, _, files in os.walk(SOURCE_DIR):

            for file in files:
                if file.lower().endswith(TARGET_EXTENSIONS):

                    total_files += 1
                    full_path = os.path.join(root, file)
                    abs_path = os.path.abspath(root)

                    try:
                        with open(full_path, 'r', encoding='utf-8', errors='ignore') as f:
                            total_file_lines += sum(1 for _ in f)
                    except:
                        pass

                    queries = extract_queries_from_file(full_path)

                    for query in queries:
                        total_queries += 1
                        crud_type = classify_query(query)

                        for line in query.splitlines():
                            if line.strip():
                                out_file.write(
                                    f"{crud_type}{DELIMITER}"
                                    f"{abs_path}{DELIMITER}"
                                    f"{file}{DELIMITER}"
                                    f"{full_path}{DELIMITER}"
                                    f"{line.rstrip()}\n"
                                )
                                total_output_rows += 1

    print("====================================")
    print(" SQL 추출 완료 (ALTER TABLE 전용 + FROM DUAL 단독만 제외)")
    print("====================================")
    print(f"DAT 파일 위치        : {output_path}")
    print(f"처리 파일 건수        : {total_files}")
    print(f"추출 쿼리 건수        : {total_queries}")
    print(f"생성 행 건수          : {total_output_rows}")
    print(f"전체 파일 총 행 건수  : {total_file_lines}")
    print("====================================")

if __name__ == "__main__":
    main()

def is_valid_table(name):
    name = name.upper().strip()
    if name in SQL_RESERVED:
        return False
    return bool(re.match(r'^[A-Z0-9_]+\.[A-Z0-9_]+$|^[A-Z0-9_]+$', name))

def extract_sql_info(query):

    sql = query.upper()

    target_tables = set()
    source_tables = set()
    sql_typ = ""

    # --------------------------
    # SQL TYPE 판별
    # --------------------------

    if sql.startswith("INSERT"):
        sql_typ = "INSERT"
        m = re.search(r'INSERT\s+INTO\s+([A-Z0-9_.]+)', sql)
        if m and is_valid_table(m.group(1)):
            target_tables.add(m.group(1))

    elif sql.startswith("UPDATE"):
        sql_typ = "UPDATE"
        m = re.search(r'UPDATE\s+([A-Z0-9_.]+)', sql)
        if m and is_valid_table(m.group(1)):
            target_tables.add(m.group(1))

    elif sql.startswith("DELETE"):
        sql_typ = "DELETE"
        m = re.search(r'DELETE\s+FROM\s+([A-Z0-9_.]+)', sql)
        if m and is_valid_table(m.group(1)):
            target_tables.add(m.group(1))

    elif sql.startswith("MERGE"):
        sql_typ = "MERGE"
        m = re.search(r'MERGE\s+INTO\s+([A-Z0-9_.]+)', sql)
        if m and is_valid_table(m.group(1)):
            target_tables.add(m.group(1))

    elif re.search(r'CREATE\s+TABLE', sql) and "AS SELECT" in sql:
        sql_typ = "CTAS"
        m = re.search(r'CREATE\s+TABLE\s+([A-Z0-9_.]+)', sql)
        if m and is_valid_table(m.group(1)):
            target_tables.add(m.group(1))

    elif sql.startswith("SELECT"):
        sql_typ = "SELECT"

    else:
        sql_typ = "ETC"

    # --------------------------
    # SOURCE TABLE 추출 (서브쿼리 포함 전체 스캔)
    # --------------------------

    for m in re.findall(r'\bFROM\s+([A-Z0-9_.]+)', sql):
        if is_valid_table(m):
            source_tables.add(m)

    for m in re.findall(r'\bJOIN\s+([A-Z0-9_.]+)', sql):
        if is_valid_table(m):
            source_tables.add(m)

    for m in re.findall(r'\bUSING\s+([A-Z0-9_.]+)', sql):
        if is_valid_table(m):
            source_tables.add(m)

    return sql_typ, target_tables, source_tables

요청하신 요구사항을 반영하여, 추출된 SQL 쿼리에서 Source Table과 Target Table을 정교하게 분리하고, 이를 CSV 형식으로 저장하는 전체 소스 코드를 작성해 드립니다.

주요 수정 및 강화 사항
SQL 파싱 로직: WITH절, CTAS(Create Table As), MERGE INTO, INSERT INTO, UPDATE, DELETE 및 서브쿼리 내부의 테이블을 추출하기 위한 정규표현식 로직을 추가했습니다.

예약어 필터링: SET, WHERE, VALUES 등 테이블명으로 오인될 수 있는 SQL 키워드를 제외 목록에 등록하여 정확도를 높였습니다.

다대다 구조 대응: 하나의 Target Table에 대해 여러 Source Table이 매핑될 경우 여러 행으로 나누어 출력합니다.

CSV 레이아웃 준수: crud_type,abs_path,file,full_path,sql_typ,source_table,target_table 형식을 유지합니다.

#!/usr/bin/env python3
# 실행방법:
# python3 프로그램명.py 절대경로포함소스디렉토리

import os
import re
import sys
import csv
from datetime import datetime

# ==============================
# 1. 파라미터 체크 및 설정
# ==============================

if len(sys.argv) != 2:
    print("사용법: python3 프로그램명.py 절대경로포함소스디렉토리")
    sys.exit(1)

SOURCE_DIR = os.path.abspath(sys.argv[1])
if not os.path.isdir(SOURCE_DIR):
    print("오류: 유효한 디렉토리가 아닙니다.")
    sys.exit(1)

TARGET_EXTENSIONS = ('.sh', '.hql', '.sql', '.uld', '.ld')
RESERVED_WORDS = {
    "SET", "WHERE", "AND", "OR", "ON", "WHEN", "THEN", "ELSE", 
    "VALUES", "SELECT", "UPDATE", "INSERT", "DELETE", "MERGE", 
    "USING", "FROM", "JOIN", "INTO", "GROUP", "ORDER", "BY", 
    "HAVING", "END", "COMMIT", "DISTINCT", "AS", "CASE", "UNION", "ALL"
}

# ==============================
# 2. 전처리 및 쿼리 추출 로직
# ==============================

def preprocess(content):
    # 주석 제거 및 클리닝
    content = re.sub(r'--.*$', '', content, flags=re.MULTILINE)
    content = re.sub(r'/\*.*?\*/', '', content, flags=re.DOTALL)
    content = "\n".join(line for line in content.splitlines() if not line.lstrip().startswith("#"))
    content = "\n".join(line for line in content.splitlines() if not re.match(r'(?i)^\s*DBMS_OUTPUT', line))
    return content

MAIN_QUERY_START = re.compile(
    r"\b(CREATE\s+(?:TEMPORARY\s+)?VIEW|CREATE\s+TABLE|ALTER\s+TABLE|DROP|TRUNCATE|MERGE|UPSERT|INSERT|UPDATE|DELETE|SELECT)\b",
    re.IGNORECASE
)

def extract_queries_from_file(file_path):
    queries = []
    try:
        with open(file_path, 'r', encoding='utf-8', errors='ignore') as f:
            content = preprocess(f.read())
        
        pos = 0
        while pos < len(content):
            match = MAIN_QUERY_START.search(content, pos)
            if not match: break
            
            start = match.start()
            # 세미콜론(;) 찾기 (문자열 내부 무시)
            end = start
            in_string = False
            while end < len(content):
                if content[end] in ("'", '"'): in_string = not in_string
                if content[end] == ";" and not in_string:
                    end += 1
                    break
                end += 1
            
            query = content[start:end].strip()
            if query: queries.append(query)
            pos = end
    except Exception as e:
        print(f"[ERROR] {file_path}: {e}")
    return queries

# ==============================
# 3. SQL 구조 분석 및 테이블 추출 (핵심)
# ==============================

def parse_tables(query):
    sql_upper = query.upper()
    sql_typ = sql_upper.split()[0] if sql_upper.split() else "UNKNOWN"
    
    targets = set()
    sources = set()

    # 1. Target Table 추출
    if "INSERT INTO" in sql_upper or "INSERT OVERWRITE TABLE" in sql_upper:
        m = re.search(r"INSERT\s+(?:INTO|OVERWRITE\s+TABLE)\s+([^\s\(\)]+)", sql_upper)
        if m: targets.add(m.group(1))
    elif "UPDATE" in sql_upper and sql_typ == "UPDATE":
        m = re.search(r"UPDATE\s+([^\s]+)", sql_upper)
        if m: targets.add(m.group(1))
    elif "MERGE INTO" in sql_upper:
        m = re.search(r"MERGE\s+INTO\s+([^\s]+)", sql_upper)
        if m: targets.add(m.group(1))
    elif "CREATE TABLE" in sql_upper:
        m = re.search(r"CREATE\s+TABLE\s+(?:IF\s+NOT\s+EXISTS\s+)?([^\s\(\)]+)", sql_upper)
        if m: targets.add(m.group(1))

    # 2. Source Table 추출 (FROM, JOIN, USING 뒤)
    # 괄호 안의 서브쿼리나 콤마로 구분된 리스트 대응을 위한 정규식
    src_matches = re.findall(r"(?:FROM|JOIN|USING)\s+([^\s\(\),;]+)", sql_upper)
    for name in src_matches:
        sources.add(name.strip().replace('"', '').replace('`', ''))

    # 3. 정제: 예약어 제거 및 DUAL 제거
    def clean_tables(table_set):
        cleaned = set()
        for t in table_set:
            t_clean = t.split('.')[-1] # 스키마 제거하고 체크
            if t.upper() not in RESERVED_WORDS and t_clean.upper() not in RESERVED_WORDS and t.upper() != "DUAL":
                # 괄호 등 특수문자 제거
                final_name = re.sub(r'[()\;]', '', t)
                if final_name: cleaned.add(final_name)
        return cleaned

    targets = clean_tables(targets)
    sources = clean_tables(sources)
    
    # Target이 Source에 포함되어 있다면 제거 (Self-referential 제외용 선택사항)
    # sources = sources - targets 

    return sql_typ, list(sources), list(targets)

def classify_query_type(sql_typ):
    c_types = ["CREATE", "INSERT", "MERGE", "UPSERT"]
    u_types = ["UPDATE", "ALTER", "REPLACE", "DROP", "TRUNCATE"]
    d_types = ["DELETE"]
    if sql_typ in c_types: return "C"
    if sql_typ in u_types: return "U"
    if sql_typ in d_types: return "D"
    return "R"

# ==============================
# 4. 메인 실행부
# ==============================

def main():
    program_name = os.path.splitext(os.path.basename(sys.argv[0]))[0]
    source_last_dir = os.path.basename(SOURCE_DIR.rstrip(os.sep))
    
    out_dir = os.path.join(os.getcwd(), "out")
    os.makedirs(out_dir, exist_ok=True)
    
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    output_filename = f"{program_name}_{source_last_dir}_{timestamp}.csv"
    output_path = os.path.join(out_dir, output_filename)

    total_files = 0
    total_rows = 0

    with open(output_path, 'w', encoding='utf-8', newline='') as csvfile:
        writer = csv.writer(csvfile, delimiter=',')
        # 헤더 작성
        writer.writerow(['crud_type', 'abs_path', 'file', 'full_path', 'sql_typ', 'source_table', 'target_table'])

        for root, _, files in os.walk(SOURCE_DIR):
            for file in files:
                if file.lower().endswith(TARGET_EXTENSIONS):
                    total_files += 1
                    full_path = os.path.join(root, file)
                    abs_path = os.path.abspath(root)
                    
                    queries = extract_queries_from_file(full_path)
                    
                    for query in queries:
                        sql_typ, sources, targets = parse_tables(query)
                        crud_type = classify_query_type(sql_typ)
                        
                        # Target이 없으면 빈값 처리, Source가 여러개면 행 분리
                        target_display = ",".join(targets) if targets else ""
                        
                        if not sources:
                            # 소스 테이블이 없는 경우 (예: INSERT INTO ... VALUES)
                            writer.writerow([crud_type, abs_path, file, full_path, sql_typ, "", target_display])
                            total_rows += 1
                        else:
                            # 소스 테이블별로 행 생성
                            for src in sources:
                                # Target이 여러개일 경우도 고려하여 루프 가능하지만 보통은 1개
                                if not targets:
                                    writer.writerow([crud_type, abs_path, file, full_path, sql_typ, src, ""])
                                    total_rows += 1
                                else:
                                    for tgt in targets:
                                        writer.writerow([crud_type, abs_path, file, full_path, sql_typ, src, tgt])
                                        total_rows += 1

    print("====================================")
    print(f" CSV 생성 완료: {output_path}")
    print(f" 처리 파일 수: {total_files}")
    print(f" 생성 데이터 행 수: {total_rows}")
    print("====================================")

if __name__ == "__main__":
    main()

Source 추출: FROM, JOIN, USING 키워드 뒤에 오는 단어들을 정규식 findall로 모두 찾아냅니다. 이는 WITH절 내부의 테이블이나 서브쿼리 외부 테이블을 모두 포함합니다.

DUAL 테이블은 관습적으로 소스 테이블에서 제외하도록 처리했습니다.

행 생성 방식:

한 쿼리에 Source A, Source B가 있고 Target T가 있다면 CSV에는 두 개의 행(A->T, B->T)이 생성됩니다.

import os
import re
import sys
import csv
from datetime import datetime

if len(sys.argv) != 2:

    sys.exit(1)

TARGET_EXTENSIONS = ('.sh', '.hql', '.sql', '.uld', '.ld')

# 테이블명으로 간주하지 않을 SQL 키워드
RESERVED_WORDS = {
    "SET", "WHERE", "AND", "OR", "ON", "WHEN", "THEN", "ELSE", "VALUES", 
    "SELECT", "UPDATE", "INSERT", "DELETE", "MERGE", "USING", "FROM", 
    "JOIN", "INTO", "GROUP", "ORDER", "BY", "HAVING", "END", "COMMIT", 
    "DISTINCT", "AS", "CASE", "UNION", "ALL", "WITH", "LEFT", "RIGHT", 
    "FULL", "INNER", "OUTER", "CROSS", "OVERWRITE", "TABLE", "IF", "NOT", "EXISTS"
}

# ==============================
# 2. 전처리 함수
# ==============================
def preprocess(content):
    # C 스타일 및 SQL 주석 제거
    content = re.sub(r'--.*$', '', content, flags=re.MULTILINE)
    content = re.sub(r'/\*.*?\*/', '', content, flags=re.DOTALL)
    # 쉘 스크립트 주석 제거 및 불필요한 구문 제거
    lines = []
    for line in content.splitlines():
        clean_line = line.lstrip()
        if not clean_line.startswith("#") and not clean_line.upper().startswith("DBMS_OUTPUT"):
            lines.append(line)
    return "\n".join(lines)

# ==============================
# 3. 분석 로직 (스키마 분리 및 특수 구문)
# ==============================
def split_schema_table(full_name):
    """DBNAME.TABLE 형태를 분리하여 (schema, table) 반환"""
    full_name = full_name.replace('"', '').replace('`', '').replace('[', '').replace(']', '')
    if '.' in full_name:
        parts = full_name.split('.')
        return parts[0], ".".join(parts[1:])
    return "", full_name

def get_table_list(query):
    sql_upper = query.upper()
    # 0. WITH 절 Alias 추출 (소스 테이블에서 제외하기 위함)
    cte_aliases = set()
    with_match = re.search(r"WITH\s+([A-Z0-9_]+)\s+AS", sql_upper)
    if with_match:
        cte_aliases.add(with_match.group(1))

    targets = set()
    sources = set()
    
    # 1. Target 추출 (INSERT, UPDATE, DELETE, MERGE, CREATE, REPLACE)
    # CTAS 및 일반 CREATE
    create_match = re.search(r"CREATE\s+(?:TEMPORARY\s+)?TABLE\s+(?:IF\s+NOT\s+EXISTS\s+)?([^\s\(\)]+)", sql_upper)
    if create_match: targets.add(create_match.group(1))
    
    # INSERT / REPLACE (MySQL/Hive)
    insert_match = re.search(r"(?:INSERT|REPLACE)\s+(?:INTO|OVERWRITE\s+TABLE)\s+([^\s\(\)]+)", sql_upper)
    if insert_match: targets.add(insert_match.group(1))
    
    # UPDATE
    update_match = re.search(r"UPDATE\s+([^\s]+)", sql_upper)
    if update_match: targets.add(update_match.group(1))
    
    # MERGE INTO (Oracle/Teradata)
    merge_match = re.search(r"MERGE\s+INTO\s+([^\s]+)", sql_upper)
    if merge_match: targets.add(merge_match.group(1))

    # DELETE
    delete_match = re.search(r"DELETE\s+(?:FROM\s+)?([^\s]+)", sql_upper)
    if delete_match: targets.add(delete_match.group(1))

    # 2. Source 추출 (FROM, JOIN, USING)
    # 서브쿼리 및 조인 포함 모든 소스 후보 추출
    src_matches = re.findall(r"(?:FROM|JOIN|USING)\s+([^\s\(\),;]+)", sql_upper)
    for name in src_matches:
        name_clean = name.strip().rstrip(';')
        # 예약어, CTE Alias, DUAL, 숫자 등 제외
        if (name_clean not in RESERVED_WORDS and 
            name_clean not in cte_aliases and 
            name_clean != "DUAL" and 
            not name_clean.isdigit()):
            sources.add(name_clean)

    return list(sources), list(targets)

def main():
    program_name = os.path.splitext(os.path.basename(sys.argv[0]))[0]

    output_path = os.path.join(out_dir, f"{program_name}_{source_last_dir}_{timestamp}.csv")

    MAIN_QUERY_START = re.compile(
        r"\b(CREATE|INSERT|REPLACE|MERGE|UPDATE|DELETE|SELECT|TRUNCATE|DROP|ALTER)\b", 
        re.IGNORECASE
    )

    total_files = 0
    total_rows = 0

    with open(output_path, 'w', encoding='utf-8', newline='') as csvfile:
        writer = csv.writer(csvfile)
        writer.writerow([
            'crud_type', 'abs_path', 'file', 'full_path', 'sql_typ', 
            'source_schema', 'source_table', 'target_schema', 'target_table'
        ])

        for root, _, files in os.walk(SOURCE_DIR):
            for file in files:
                if file.lower().endswith(TARGET_EXTENSIONS):
                    total_files += 1
                    full_path = os.path.join(root, file)

                        with open(full_path, 'r', encoding='utf-8', errors='ignore') as f:
                            content = preprocess(f.read())
                        
                        pos = 0
                        while pos < len(content):
                            match = MAIN_QUERY_START.search(content, pos)
                            if not match: break
                            
                            start = match.start()
                            end = start
                            in_string = False
                            while end < len(content):
                                if content[end] in ("'", '"'): in_string = not in_string
                                if content[end] == ";" and not in_string:
                                    end += 1
                                    break
                                end += 1
                            
                            query = content[start:end].strip()
                            if query:
                                sql_typ = query.split()[0].upper()
                                sources, targets = get_table_list(query)
                                
                                # CRUD 타입 결정

                                elif sql_typ in ("UPDATE", "ALTER"): crud = "U"

                            pos = end

    print("-" * 40)

로직 강화 상세 설명
CTAS (Create Table As Select):

CREATE TABLE ... 구문에서 테이블명을 Target으로 잡고, 이후 SELECT 문에 나오는 테이블들을 모두 Source로 추출합니다.

WITH (CTE) 필터링:

WITH TMP AS (SELECT ...) 구문 사용 시 TMP는 물리 테이블이 아니므로 소스 목록에서 자동으로 걸러냅니다.

스키마 분리 로직:

split_schema_table 함수가 OWNER.TABLE_NAME을 점(.) 기준으로 쪼개어 각각의 컬럼에 배정합니다. 스키마가 없는 경우 공백으로 처리됩니다.

다중 소스/타겟 대응:

RESERVED_WORDS = {
    "SET", "WHERE", "AND", "OR", "ON", "WHEN", "THEN", "ELSE", "VALUES", 
    "SELECT", "UPDATE", "INSERT", "DELETE", "MERGE", "USING", "FROM",

ONLY_FROM_DUAL_PATTERN = re.compile(r'^\s*SELECT\s+.*?\s+FROM\s+DUAL\s*;?\s*$', re.IGNORECASE | re.DOTALL)

# ==============================
# 2. 전처리 및 스키마 분리

def preprocess(content):
    content = re.sub(r'--.*$', '', content, flags=re.MULTILINE)
    content = re.sub(r'/\*.*?\*/', '', content, flags=re.DOTALL)
    lines = [line for line in content.splitlines() if not line.lstrip().startswith("#") and not re.match(r'(?i)^\s*DBMS_OUTPUT', line)]
    return "\n".join(lines)

def split_schema_table(full_name):
    if not full_name: return "", ""
    full_name = re.sub(r'[()\[\]`"\;]', '', full_name)
    if '.' in full_name:
        parts = full_name.split('.')
        return parts[0], ".".join(parts[1:])
    return "", full_name

# ==============================
# 3. 테이블 추출 엔진
# ==============================
def get_table_info(query):
    sql_upper = query.upper().strip()
    
    # [제외조건 1] FROM DUAL 단독 쿼리
    if ONLY_FROM_DUAL_PATTERN.match(sql_upper):
        return None, [], []

    sql_typ = sql_upper.split()[0] if sql_upper.split() else "UNKNOWN"
    
    # CTE(WITH절) Alias 수집
    cte_aliases = set(re.findall(r"WITH\s+([A-Z0-9_]+)\s+AS", sql_upper))

    targets = set()
    sources = set()

    # --- Target 추출 로직 ---
    # 1. CREATE / INSERT / REPLACE / MERGE / UPDATE
    m_tgt = re.search(r"(?:CREATE|INSERT|REPLACE|MERGE|UPDATE)\s+(?:INTO|OVERWRITE\s+TABLE|TABLE\s+)?([^\s\(\),;]+)", sql_upper)
    if m_tgt: targets.add(m_tgt.group(1))

    # 2. DELETE FROM [TABLE] 추출 강화
    m_del = re.search(r"DELETE\s+(?:FROM\s+)?([^\s\(\),;]+)", sql_upper)
    if m_del: targets.add(m_del.group(1))

    # 3. ALTER TABLE / TRUNCATE TABLE 추출
    m_alt_tru = re.search(r"(?:ALTER|TRUNCATE)\s+TABLE\s+([^\s\(\),;]+)", sql_upper)
    if m_alt_tru: targets.add(m_alt_tru.group(1))

    # [제외조건 2] ALTER로 시작하는데 타겟/소스 테이블이 없으면 무시
    if sql_typ == "ALTER" and not targets:
        return None, [], []

    # --- Source 추출 로직 ---
    # FROM, JOIN, USING 뒤의 테이블 추출
    src_matches = re.findall(r"(?:FROM|JOIN|USING)\s+([^\s\(\),;]+)", sql_upper)
    for name in src_matches:
        name_clean = name.strip()
        if (name_clean not in RESERVED_WORDS and 
            name_clean not in cte_aliases and 
            name_clean != "DUAL" and 
            not name_clean.isdigit()):
            # DELETE 구문에서 타겟으로 이미 잡힌 것은 소스에서 제외 (선택 사항)
            sources.add(name_clean)

    return sql_typ, list(sources), list(targets)

보완된 로직 상세 설명
INSERT OVERWRITE 정교화:

기존 정규식에서 단순히 INTO나 OVERWRITE를 찾는 것이 아니라, INSERT OVERWRITE TABLE [TABLE] 혹은 INSERT OVERWRITE [TABLE] 뒤의 단어를 정확히 캡처하도록 그룹핑을 수정했습니다.

UPDATE FROM 분석:

예시로 주신 WITH (...) UPDATE table_target SET ... FROM table_source 구문의 경우:

UPDATE 뒤의 table_target을 Target으로 추출합니다.

WITH 내부의 FROM table_1과 SET 이후의 FROM table_2를 re.findall을 통해 모두 Source로 수집합니다.

DROP INDEX 필터링:

쿼리 시작이 DROP INDEX인 경우 분석을 스킵하여 인덱스 명이 테이블로 오인되는 것을 방지했습니다.

스키마 분리 안정화:

split_schema_table 함수 내에서 추출된 단어가 RESERVED_WORDS(INTO, OVERWRITE 등)인 경우 빈 값을 반환하게 하여 잘못된 데이터가 CSV에 들어가는 것을 한 번 더 차단했습니다.

# 2. 유틸리티 함수

def preprocess(content):
    content = re.sub(r'--.*$', '', content, flags=re.MULTILINE)
    content = re.sub(r'/\*.*?\*/', '', content, flags=re.DOTALL)
    lines = [line for line in content.splitlines() if not line.lstrip().startswith("#") and not re.match(r'(?i)^\s*DBMS_OUTPUT', line)]
    return "\n".join(lines)

def split_schema_table(full_name):
    if not full_name: return "", ""
    # 특수문자 제거 및 정리
    full_name = re.sub(r'[()\[\]`"\;]', '', full_name).strip()
    if not full_name or full_name.upper() in RESERVED_WORDS: return "", ""
    
    if '.' in full_name:
        parts = full_name.split('.')
        return parts[0], ".".join(parts[1:])
    return "", full_name

# ==============================
# 3. 테이블 추출 엔진 (핵심 로직 보강)
# ==============================
def get_table_info(query):
    sql_upper = query.upper().strip()
    
    # [제외조건 1] FROM DUAL 단독 쿼리
    if ONLY_FROM_DUAL_PATTERN.match(sql_upper):
        return None, [], []
    
    # [제외조건 2] DROP INDEX 제외
    if sql_upper.startswith("DROP INDEX") or sql_upper.startswith("DROP\nINDEX"):
        return None, [], []

    sql_typ = sql_upper.split()[0] if sql_upper.split() else "UNKNOWN"
    
    # CTE(WITH절) Alias 수집 - 소스에서 제외하기 위함
    cte_aliases = set(re.findall(r"WITH\s+([A-Z0-9_]+)\s+AS", sql_upper))

    targets = set()
    sources = set()

    # --- 1. Target 추출 ---
    # INSERT INTO / INSERT OVERWRITE (TABLE) 패턴
    insert_match = re.search(r"INSERT\s+(?:INTO|OVERWRITE(?:\s+TABLE)?)\s+([^\s\(\),;]+)", sql_upper)
    if insert_match:
        targets.add(insert_match.group(1))

    # UPDATE 패턴 (UPDATE [TABLE] SET ...)
    update_match = re.search(r"UPDATE\s+([^\s\(\),;]+)", sql_upper)
    if update_match:
        targets.add(update_match.group(1))

    # MERGE INTO / DELETE FROM / CREATE TABLE / TRUNCATE TABLE / ALTER TABLE
    others_match = re.search(r"(?:MERGE\s+INTO|DELETE(?:\s+FROM)?|CREATE(?:\s+TEMPORARY)?\s+TABLE|TRUNCATE\s+TABLE|ALTER\s+TABLE)\s+([^\s\(\),;]+)", sql_upper)
    if others_match:
        targets.add(others_match.group(1))

    # [제외조건 3] ALTER로 시작하는데 타겟 테이블이 없으면 무시
    if sql_typ == "ALTER" and not targets:
        return None, [], []

    # --- 2. Source 추출 (FROM, JOIN, USING) ---
    # UPDATE ... FROM 이나 SELECT ... FROM 등의 모든 소스 후보군 추출
    src_matches = re.findall(r"(?:FROM|JOIN|USING)\s+([^\s\(\),;]+)", sql_upper)
    for name in src_matches:
        name_clean = name.strip()
        if (name_clean not in RESERVED_WORDS and 
            name_clean not in cte_aliases and 
            name_clean != "DUAL" and 
            not name_clean.isdigit()):
            sources.add(name_clean)

    # 타겟으로 선정된 테이블이 소스에 중복 포함된 경우 제거 (필요 시 주석 처리)
    # sources = sources - targets

    return sql_typ, list(sources), list(targets)

    if delete_match:

#!/usr/bin/env python3
import os
import re
import sys
import csv
from datetime import datetime

# ==============================
# 1. 설정
# ==============================
if len(sys.argv) != 2:
    print("사용법: python3 프로그램명.py 절대경로포함소스디렉토리")
    sys.exit(1)

SOURCE_DIR = os.path.abspath(sys.argv[1])
TARGET_EXTENSIONS = ('.sh', '.hql', '.sql', '.uld', '.ld')

RESERVED_WORDS = {
    "SET","WHERE","AND","OR","ON","WHEN","THEN","ELSE","VALUES",
    "SELECT","UPDATE","INSERT","DELETE","MERGE","USING","FROM",
    "JOIN","INTO","GROUP","ORDER","BY","HAVING","END","COMMIT",
    "DISTINCT","AS","CASE","UNION","ALL","WITH","LEFT","RIGHT",
    "FULL","INNER","OUTER","CROSS","OVERWRITE","TABLE","IF",
    "NOT","EXISTS","INDEX"
}

ONLY_FROM_DUAL_PATTERN = re.compile(
    r'^\s*SELECT\s+.*?\s+FROM\s+DUAL\s*;?\s*$',
    re.IGNORECASE | re.DOTALL
)

# ==============================
# 2. 전처리
# ==============================
def preprocess(content):
    content = re.sub(r'--.*$', '', content, flags=re.MULTILINE)
    content = re.sub(r'/\*.*?\*/', '', content, flags=re.DOTALL)
    return content


# ==============================
# 3. schema.table 분리 (alias.column 차단)
# ==============================
def split_schema_table(name):

    if not name:
        return "", ""

    name = re.sub(r'[()\[\]`"\;]', '', name).strip()

    if not name:
        return "", ""

    # alias.column 형태 차단
    if name.count('.') == 1:
        left, right = name.split('.')
        # alias는 보통 1~3자 소문자 → heuristic 차단
        if len(left) <= 3:
            return "", ""   # 완전 차단

    if '.' in name:
        parts = name.split('.')
        return parts[0], ".".join(parts[1:])

    return "", name


# ==============================
# 4. 테이블 추출 엔진 (완전 수정)
# ==============================
def get_table_info(query):

    sql = query
    sql_upper = sql.upper().strip()

    if ONLY_FROM_DUAL_PATTERN.match(sql_upper):
        return None, [], []

    if sql_upper.startswith("DROP INDEX"):
        return None, [], []

    sql_typ = sql_upper.split()[0] if sql_upper.split() else None

    targets = set()
    sources = set()
    alias_map = {}

    # ============================
    # TARGET 추출
    # ============================

    patterns = [
        r"INSERT\s+(?:INTO|OVERWRITE(?:\s+TABLE)?)\s+([^\s\(\),;]+)",
        r"UPDATE\s+([^\s\(\),;]+)",
        r"DELETE\s+FROM\s+([^\s\(\),;]+)",
        r"MERGE\s+INTO\s+([^\s\(\),;]+)",
        r"(?:CREATE(?:\s+TEMPORARY)?\s+TABLE|TRUNCATE\s+TABLE|ALTER\s+TABLE)\s+([^\s\(\),;]+)"
    ]

    for p in patterns:
        m = re.search(p, sql_upper)
        if m:
            targets.add(m.group(1))

    # ============================
    # FROM 절 정확 추출
    # ============================

    from_match = re.search(
        r"\bFROM\b(.*?)(?=\bWHERE\b|\bGROUP\b|\bORDER\b|\bHAVING\b|\bUNION\b|;|$)",
        sql,
        re.IGNORECASE | re.DOTALL
    )

    if from_match:
        from_block = from_match.group(1)

        table_parts = from_block.split(',')

        for part in table_parts:
            part = part.strip()
            if not part:
                continue

            tokens = part.split()

            table_name = tokens[0]
            alias = None

            if len(tokens) > 1:
                if tokens[1].upper() != "AS":
                    alias = tokens[1]
                elif len(tokens) > 2:
                    alias = tokens[2]

            sources.add(table_name)

            if alias:
                alias_map[alias.upper()] = table_name.upper()

    # ============================
    # JOIN 절 처리
    # ============================

    join_matches = re.findall(
        r"\bJOIN\s+([^\s\(\),;]+)",
        sql_upper
    )

    for j in join_matches:
        sources.add(j)

    # ============================
    # alias.column 완전 차단
    # ============================

    clean_sources = set()

    for s in sources:
        s_clean = s.strip()

        # alias 자체 제거
        if s_clean.upper() in alias_map:
            continue

        # alias.column 제거
        if '.' in s_clean:
            left = s_clean.split('.')[0]
            if left.upper() in alias_map:
                continue

        clean_sources.add(s_clean)

    return sql_typ, list(clean_sources), list(targets)


# ==============================
# 5. 메인
# ==============================
def main():

    program_name = os.path.splitext(os.path.basename(sys.argv[0]))[0]
    source_last_dir = os.path.basename(SOURCE_DIR.rstrip(os.sep))
    out_dir = os.path.join(os.getcwd(), "out")
    os.makedirs(out_dir, exist_ok=True)

    output_path = os.path.join(
        out_dir,
        f"{program_name}_{source_last_dir}_{datetime.now().strftime('%Y%m%d_%H%M%S')}.csv"
    )

    QUERY_START_PATTERN = re.compile(
        r"\b(CREATE|INSERT|REPLACE|MERGE|UPDATE|DELETE|SELECT|TRUNCATE|DROP|ALTER)\b",
        re.IGNORECASE
    )

    total_files = 0
    total_rows = 0

    with open(output_path, 'w', encoding='utf-8', newline='') as csvfile:

        writer = csv.writer(csvfile)
        writer.writerow([
            'crud_type','abs_path','file','full_path',
            'sql_typ','source_schema','source_table',
            'target_schema','target_table'
        ])

        for root, _, files in os.walk(SOURCE_DIR):

            for file in files:

                if file.lower().endswith(TARGET_EXTENSIONS):

                    total_files += 1
                    full_path = os.path.join(root, file)
                    abs_path = os.path.abspath(root)

                    try:
                        with open(full_path, 'r', encoding='utf-8', errors='ignore') as f:
                            content = preprocess(f.read())

                        pos = 0

                        while pos < len(content):

                            match = QUERY_START_PATTERN.search(content, pos)
                            if not match:
                                break

                            start = match.start()
                            end = start
                            in_string = False

                            while end < len(content):
                                if content[end] in ("'", '"'):
                                    in_string = not in_string
                                if content[end] == ";" and not in_string:
                                    end += 1
                                    break
                                end += 1

                            query = content[start:end].strip()

                            if query:

                                sql_typ, sources, targets = get_table_info(query)

                                if sql_typ:

                                    crud = "R"
                                    if sql_typ in ("INSERT","CREATE","REPLACE","MERGE"):
                                        crud = "C"
                                    elif sql_typ in ("UPDATE","ALTER","TRUNCATE"):
                                        crud = "U"
                                    elif sql_typ in ("DELETE","DROP"):
                                        crud = "D"

                                    for t in targets if targets else [""]:
                                        t_schema, t_table = split_schema_table(t)

                                        for s in sources if sources else [""]:
                                            s_schema, s_table = split_schema_table(s)

                                            if s_table == "" and s != "":
                                                continue

                                            writer.writerow([
                                                crud, abs_path, file, full_path,
                                                sql_typ,
                                                s_schema, s_table,
                                                t_schema, t_table
                                            ])
                                            total_rows += 1

                            pos = end

                    except Exception as e:
                        print(f"오류 ({file}): {e}")

    print("\n[추출 완료]")
    print(f"결과 저장: {output_path}")
    print(f"파일 수: {total_files}, 생성 행 수: {total_rows}")


if __name__ == "__main__":
    main()
