# 3D 테트리스 게임 - 실행 가이드

## 프로그램 개요
이 프로그램은 Python으로 만든 3D 테트리스 게임입니다. Tkinter를 사용하여 구현되었으므로 특별한 외부 라이브러리 설치가 필요 없습니다.

## 파일 정보
- **파일명**: `tetris_3d.py`
- **위치**: `d:\workspace\yys_test01\`
- **언어**: Python 3.x
- **필수 라이브러리**: 표준 라이브러리 (tkinter, math, random)

## 시스템 요구사항
- Python 3.6 이상
- Windows / Mac / Linux
- 최소 해상도: 900x700 픽셀

## 실행 방법

### 방법 1: Python 직접 실행 (권장)
```bash
python tetris_3d.py
```

## 게임 조작법

### 키보드 조작
| 키 | 기능 |
|---|---|
| **← →** (좌우 화살표) | 블록 좌우 이동 |
| **↑ ↓** (상하 화살표) | 블록 상하 이동 |
| **W** | X축 기준 회전 |
| **A** | Y축 기준 회전 |
| **S** | Z축 기준 회전 |
| **Q** | 게임 종료 |

### 게임 규칙
1. 다양한 모양의 3D 테트로미노 블록이 아래로 떨어집니다
2. 방향키로 블록을 이동시킬 수 있습니다
3. W, A, S 키로 3D 회전을 할 수 있습니다
4. 블록이 바닥에 닿으면 고정되고 새로운 블록이 생성됩니다
5. 블록이 고정될 때마다 10점씩 획득합니다

## 프로그램 구조

### 주요 클래스

#### `Point3D` 클래스
- 3D 공간의 점을 표현합니다
- X, Y, Z 좌표를 가집니다
- 3축(X, Y, Z) 회전 메서드 제공
- 2D 평면에 투영하는 메서드 제공

#### `Tetromino3D` 클래스
- 테트로미노 블록을 표현합니다
- 7가지 기본 테트로미노 형태 제공:
  - **I형**: 직선 모양
  - **O형**: 정사각형 모양
  - **T형**: T자 모양
  - **L형**: L자 모양
  - **J형**: 역L자 모양
  - **S형**: S자 모양
  - **Z형**: Z자 모양
- 회전, 이동 기능 제공

#### `Game3DTetris` 클래스
- 게임의 메인 로직을 담당합니다
- 현재 블록과 떨어진 블록들 관리
- 점수 관리
- 화면 렌더링
- 게임 루프 실행

## 기술 상세

### 3D 투영
프로그램은 **원근 투영(Perspective Projection)**을 사용합니다:
```
scale = distance / (distance + z)
2d_x = 3d_x * scale + screen_width/2
2d_y = -3d_y * scale + screen_height/2
```

### 회전 변환
3D 회전은 **회전 행렬(Rotation Matrix)**을 사용합니다:
- **X축 회전**: Y, Z 좌표를 회전
- **Y축 회전**: X, Z 좌표를 회전
- **Z축 회전**: X, Y 좌표를 회전

## 게임 팁

1. **블록 회전 활용**: W, A, S 키로 블록을 다양하게 회전시켜 최적의 위치에 배치하세요
2. **빠른 이동**: 방향키를 빠르게 눌러 블록을 신속히 이동시킬 수 있습니다
3. **점수 획득**: 블록이 고정될 때마다 10점을 획득하므로 계속 진행하면 점수가 증가합니다
4. **3D 효과**: 원근감 있는 3D 표현으로 더 생생한 게임 경험을 즐길 수 있습니다

## 문제 해결

### 게임이 실행되지 않는 경우
1. Python이 정상 설치되었는지 확인: `python --version`
2. tkinter 설치 확인 (대부분의 Python 배포판에 포함)
3. 파일 경로 확인

### 게임이 느린 경우
- 게임 설정에서 `fall_speed` 값 감소 (기본값: 30ms)
- 시스템 리소스 확인

### 키 입력 반응 없음
- 게임 창이 활성화되어 있는지 확인
- 다른 프로그램에서 키 입력을 가로채는지 확인
