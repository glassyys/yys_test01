# 🌱 실증 사업 대비 AI 모델 예제 프로젝트

본 프로젝트는 **AI 허브 데이터 기반 농업 AI 모델 실증 사업 대비 예제**로,  
이미지·NDVI·기상·토양 데이터의 **멀티모달 학습 및 앙상블 구조**를 검토·구현하는 것을 목표로 합니다.

---

## 📌 프로젝트 개요

- **목적**
  - 실증 사업 제출을 위한 AI 모델 구조 검증
  - 멀티모달 데이터 융합(Fusion / Ensemble) 방식 비교
  - 추후 병해충 모델 및 XAI 확장 가능 구조 설계

- **기준 일자**
  - 2026-01-12

## 🧠 모델 설계 방향

### 1️⃣ 이미지 모델
- **Base Model**
  - ResNet-50
- **역할**
  - 작물 이미지 기반 특징 추출
  - 병해충 관련 시각적 패턴 학습

---

### 2️⃣ 멀티모달 데이터 처리 레이어

- **NDVI 데이터 처리 레이어**
  - 시계열 또는 통계 기반 Feature Encoding

- **날씨 및 토양 데이터 레이어**
  - 기상·토양 정보 Feature Vector화
  - 이미지 모델과 결합 가능한 형태로 설계

---

## 🔀 모델 결합 전략 검토

### 🔹 Fusion Layer 방식
- Image Feature + NDVI + Weather + Soil
- 하나의 통합 모델로 학습

**장점**
- 데이터 간 상호작용 학습 가능

**검토 사항**
- 데이터 정합성
- 학습 난이도 및 비용

---

### 🔹 Ensemble 방식 (대안 구조)

- 개별 모델 구성
  - `ImageModel`
  - `NDVIModel`
  - `WeatherSoilModel`

- 예측 결과를 앙상블로 결합

**검토 포인트**
- Fusion Layer 대비 성능 향상 여부
- 유지보수 및 확장성

---

## 🌳 XGBoost 적용 검토

### 🔹 XGBoost Wrapper
- 딥러닝 모델 출력 Feature 또는 예측 결과를 입력
- 최종 분류/판단 모델로 사용

**고려 사항**
- 비용 대비 효과
- 개발 경험 부족 상황에서 적용 가능성
- 실증 사업 평가 관점에서의 실효성

---

## 🧪 연구·검토 중인 주요 질문

  👉 **병해충 전용 모델을 추가하여 앙상블로 결합했을 때 성능 향상 여부**
- 단일 멀티모달 모델 vs 앙상블 구조의 실증 사업 적합성
